Internet Computer - the next stage of the Internet.

DFINITY Foundation's Internet Computer (IC) is a progressive blockchain, which hosts unlimited data and computational power right "on-chain". Build scalable Dapps, DeFi, websites and much more.

Stay up to date any time with our info page!

What is a Blockchain?

Blockchain is a growing technology that allows for the storage and sharing of data in a way that is verifiable, immutable, and decentralized.
The blockchain is a distributed ledger of information, which is stored in a public database, represented as blocks, which are connected to each other by hashes; each block contains a cryptographic hash of the previous block.
